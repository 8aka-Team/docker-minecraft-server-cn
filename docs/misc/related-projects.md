# 相关项目

轻量级的Minecraft Java服务器多路复用器/代理。提供独立应用程序和Docker镜像。

### [itzg/mc-proxy](https://github.com/itzg/docker-bungeecord/)

运行由Bungeecord、Velocity或Waterfall驱动的代理的Docker镜像。

### [itzg/mc-backup](https://github.com/itzg/docker-mc-backup)
